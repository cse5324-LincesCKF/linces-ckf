# Linces’CKF — Frontend UI 💠

This is the **Premium React 19** frontend for the Linces’CKF silk platform. Designed with a **Web3 Minimalist** aesthetic, focusing on luxury, performance, and bilingual accessibility.

## 🚀 Tech Stack
- **Framework:** React 19 (Latest)
- **Build Tool:** Vite 8 + Oxc
- **Styling:** Premium Glassmorphism (Custom CSS)
- **Testing:** Vitest + React Testing Library
- **Architecture:** Role-based views (Consumer, B2B, Admin)

## 🌐 Bilingual Support
The application supports **English (EN)** and **Spanish (ES)**. 
- Content toggles are managed via local state in `App.tsx` (to be moved to Context API in Sprint 2).

## 🛠️ Getting Started
Ensure the backend is running at `http://localhost:3000`.

```bash
# Install dependencies
npm install

# Start dev server with HMR
npm run dev

# Run unit tests
npm run test
